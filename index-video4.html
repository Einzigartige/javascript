JavaScript Arrays — Interactive Playground

Key: Ordered, zero-indexed, dynamic, heterogeneous, typeof -> "object"

This interactive playground demonstrates arrays in JavaScript. Use the controls to create arrays, inspect by index, add/remove elements, and run common array methods (map, filter, reduce, etc.).
