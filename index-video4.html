JavaScript Arrays - Interactive Playground

Key: Ordered, zero-indexed, dynamic, heterogeneous, typeof -> "object"

This interactive playground demonstrates arrays in JavaScript. Use the controls to create arrays, inspect by index, add/remove elements, and run common array methods (map, filter, reduce, etc.).

Current array (variable: arr)
Access by zero-based index (arr[0] is first item).
pop() removes from end, shift() from start, unshift(v) adds at start, splice(start,deleteCount,items...) edits array.
Methods / Helpers
map() - transform
Applies a function to each element (here: doubles numbers).
filter() - select
Keeps elements that match a condition (e.g. > value).
reduce() - accumulate
Reduces array to a single value (here: sum of numbers).
find() - first match
Returns first element that matches the condition.
includes() / indexOf() - search
Check presence or index of a value.
slice() / splice() - copy/edit
slice returns a subarray, splice edits the array in-place.
sort() / reverse() - order
Sorts array (numeric if numbers), or reverses order.
join() - string
Concatenates elements into a string with a separator.
concat() - merge
Returns a new array merged with another array (does not mutate).
flat() - flatten
Flattens nested arrays by 1 level (or specified depth).
flatMap() - map + flatten
Maps then flattens result by one level.
every() - all match?
Returns true if all elements satisfy the test.
some() - any match?
Returns true if any element satisfies the test.
fill() - fill values
Fills array with a static value (mutates array).
copyWithin() - copy inside
Copies a sequence of elements to another position in the same array.
findIndex() - match index
Returns index of first element matching condition (or -1).
lastIndexOf() - last occurrence
Finds last index of a value (or -1 if not found).
reduceRight() - reduce from right
Like reduce, but processes elements from right to left.
keys()/values()/entries() - iterators
Return iterators for keys, values, or [key, value] pairs.
toString() / Array.from() - convert/create
Convert to string / create arrays from array-like values.
utilities
Useful checks and basic info.
Output / Inspector
Shows the array contents with indexes and types for clarity.
